## Финтех школа JavaScript весна 2018

### Как выполнять задачи

[См. презентацию](https://docs.google.com/presentation/d/19SpzqsVYM3L0121UaxpD-zKZdSeKzQmxthMVlGWiCSk/edit?usp=sharing)

### Установка

```
    npm install
```

### Запуск тестов

```
    npm run test
```

### Запуск линтера

```
    npm run lint
```

или

```
    npm run lint:fix
```
